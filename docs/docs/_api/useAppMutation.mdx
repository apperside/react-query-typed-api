# useAppMutation
The `useAppMutation` hook will take in input the same parameters as `useAppQuery()`.


:::note 

While useAppQuery works in the same exact way as in react-query, on useAppMutation [some little change](/docs/usage/path-params#diff-note) has been added to mutate and mutateAsync in order to give best support for path variables.

:::
